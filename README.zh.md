## 特性

- 🎯 **直观易用** - 采用VueUse风格的API设计，对Vue开发者友好
- 💪 **类型安全** - 使用TypeScript编写，提供完整的TS支持
- 🏪 **模块化** - 支持Tree-shake的ESM模块
- 🌐 **功能全面** - 完整支持Cesium常用功能
- 🎮 **交互丰富** - 提供丰富的地图元素交互能力
- ⚡️ **轻量高效** - 无冗余依赖

## 安装

```bash
# npm
npm i cesium @vueuse/core vesium

# yarn
yarn add cesium @vueuse/core vesium

# pnpm
pnpm add cesium @vueuse/core vesium
```
